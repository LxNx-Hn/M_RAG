- 문서 기준 2026-04-25

## 문서 처리

- PDF 업로드 지원
- DOCX 업로드 지원
- TXT 업로드 지원
- 업로드 크기 제한 적용
- 확장자와 MIME 검증 적용
- 사용자별 컬렉션 네임스페이스 분리
- 문서 유형 자동 분류 `paper`, `lecture`, `patent`, `general`
- 섹션 메타데이터 생성
